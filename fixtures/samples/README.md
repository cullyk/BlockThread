# fixtures/samples

This directory holds sample clothing photo inputs for development and testing.

## Important

Do **not** commit real photos of branded, copyrighted, or licensed clothing.
Only use:
- Solid-colour test garments
- Garments you own and have the right to reproduce
- Synthetic/generated test images

## Contents

- Empty for Phase 0 (scaffold only)
- Future: small synthetic test input JPEGs for the Python image pipeline

## File naming convention

`{garment_type}_{side}_{description}.{ext}`

Examples:
- `shirt_front_plain_white.jpg`
- `pants_front_blue_jeans_test.png`
