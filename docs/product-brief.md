# BlockThread – Product Brief

## Problem

Roblox creators who want to wear or sell classic clothing based on real garments currently have no reliable, fast pipeline to convert a photo of clothing into a properly laid-out 585×559 classic clothing template PNG.

## Solution

BlockThread is a web-based creator tool that:

1. Accepts front and back photos of a garment (shirt or pants).
2. Maps the garment's visual regions onto the canonical Roblox classic template canvas.
3. Exports a ready-to-upload Roblox classic clothing PNG.

## Target user

Roblox clothing designers who want to recreate real-world garments as classic clothing items.

## Scope (MVP)

- Classic shirt template
- Classic pants template
- Upload front and back photos
- Deterministic placeholder template generation
- Preview and PNG export

## Out of scope (MVP)

- Roblox layered clothing (Avatar 3.0 / UGC)
- 3D mesh generation
- Marketplace auto-publishing
- AI-powered style transfer
- User accounts / cloud storage

## Success criteria

A user can upload two clothing photos, see a labelled preview of the classic template canvas, and download a PNG that meets Roblox upload size requirements.
